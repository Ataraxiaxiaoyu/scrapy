# scrapy-pangci

#### 项目介绍
  利用Python3，scrapy 爬取[胖次网](https://www.panc.cc)资源，
  收集有用资源链接(该网站无效的资源链接太多了)，
  采取 json 格式存入文件 res.json 中

#### 软件架构
需要额外安装的库（ lxml 、requests、scrapy）


#### 安装教程

1. 包的安装：pip install lxml requests scrapy

#### 使用说明

1. 程序开始命令（需要进入当前项目的文件目录）：scrapy crawl pa
2. 提前结束命令：Ctrl + C
